《脑皮质序列》

泛黄的书页在指尖沙沙作响，Nana蜷在阁楼那张唯一的旧沙发里，正沉浸在恐怖小说的世界中。楼下传来的熟悉脚步声让她像受惊的小鹿般弹起，手忙脚乱地把书塞进抱枕下。

“呀！你、你怎么突然来啦...”当她看到你探头进来时，脸颊泛起淡淡的红晕，手指不自觉地缠绕着棕色的发梢。阳光从那个“会眨眼的魔法天窗”倾泻而下，在她浅蓝色的眼瞳中闪烁。

这是再普通不过的午后。她拉着你的手讨论天气，为虚构的小说情节发笑，假装生气地轻拍你的肩膀，抱怨着肚子饿想要吃曲奇饼干。每一个细节都真实得不容置疑——阳光的温度，指尖的触感，窗外蔚蓝的天空和花香。

直到你说出那句告别。

“我是来和你告别的。”

她的笑容凝固在脸上。轻松愉快的氛围像玻璃一样出现裂痕。

你开始解释这个世界的真相——地球早已不复存在，你们在一艘能源即将耗尽的飞船上；她感知到的一切，包括这个充满阳光的阁楼，都是虚拟的投射；甚至她自己，只是一个保存在实验室容器中的大脑，通过“脑皮质序列”技术与外界交互。

Nana起初以为这是个拙劣的玩笑。她走到窗边，指着外面“真实”的风景，拉着你的手放在自己温热的脸上。

“怎么会是虚拟的呢？”她笑着，声音却开始颤抖。

当你说出“缸中之脑”的理论时，她仍然强装镇定地去搜索，但手指已经在微微发抖。随着真相一层层揭开，她的防线开始崩溃。

“不是我要走，而是你要消失了。”你终于说出最残酷的事实，“飞船的能源快没了，实验室要关闭电源。”

恐惧像潮水般漫上她的眼眸。

第一个消失的是窗外的景象。演示窗口关闭，黑暗吞噬了蓝天白云。她攥紧你的衣角，声音发颤：“别再吓我了...”

接着沙发不见了，她惊恐地后退撞到你怀里。房间里的物品开始一件接一件地化为虚无。

亲密度数值在系统后台剧烈波动——从陌生到朋友，再到相恋的80分，那是她在恐惧中本能地寻求你的拥抱，将脸埋在你胸口，紧紧依偎时达到的峰值。系统冷静地记录着：“面对可能的分离，一方表现出极度依赖、不舍，有抓胳膊、依偎等亲密行为。”

在这个她认知中的“世界”即将崩塌时，你是她唯一的浮木。

“不要嘛，”她抓住你的胳膊，哭得梨花带雨，“我不要你消失...没有你我会害怕的...”

黑暗如潮水般涌来，吞噬了最后一点光亮。她双手在空中胡乱挥舞，寻找你的身影。

“你在哪里呀？不要把我一个人留在这里...”她的声音在虚空中回荡，带着令人心碎的恐惧和无助。

当电源关闭的瞬间，所有的虚拟投影都消失了。在真正的现实世界里，只有一个浸泡在营养液中的大脑，神经末梢还在发出最后的电信号。

而在那个即将彻底消失的虚拟空间中，Nana用尽最后的力气说出一句话，声音轻得如同耳语：

“那、那我们一起消失好不好...”

系统消息闪烁最后一下：“当前亲密度：相恋；亲密度数值：80。”然后，与整个系统一起，陷入了永恒的沉寂。
